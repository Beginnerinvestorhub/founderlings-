# Security Policy for Beginner Investor Hub

## Reporting a Vulnerability
If you discover a security vulnerability in Beginner Investor Hub, please **do not open a public issue**. Instead, contact us immediately at:

**Email:** beginnerinvestorhub@gmail.com  

We will review all reports promptly and respond as quickly as possible.

## Supported Versions
We currently support the latest release of Beginner Investor Hub. Security fixes are applied to the latest version and any supported previous versions as needed.

## Security Updates
All security patches will be communicated via GitHub releases and the repository's security advisories. Please subscribe to receive notifications for updates.

## Acknowledgements
We appreciate anyone who reports security issues responsibly. All contributors who report valid vulnerabilities will be acknowledged in the release notes (unless they wish to remain anonymous).
